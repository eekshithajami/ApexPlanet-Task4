 Student Portfolio Hub

📌 Overview

Student Portfolio Hub is a responsive web application developed as part of the **ApexPlanet Task-4** project. It showcases a student's portfolio while demonstrating core front-end web development concepts using **HTML, CSS, and JavaScript**.

The project combines multiple interactive features into a single website, including a portfolio, notes application, product listing with filtering and sorting, and a contact form with validation.



🚀 Features

* Responsive Navigation Bar
* Hero Section with Smooth Scrolling
* About Me Section
* Skills Showcase
* Project Cards
* Contact Form with JavaScript Validation
* Notes App using Local Storage
* Product Listing with:

  * Category Filtering
  * Price Sorting (Low to High / High to Low)
* Responsive Design for Desktop, Tablet, and Mobile


🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API



## 📖 Modules

### Home

Displays a welcome section with an introduction and a button for smooth navigation.

### About

Provides a brief introduction about the student and career interests.

### Skills

Displays technical skills in responsive cards with hover animations.

### Projects

Highlights completed ApexPlanet tasks.

### Notes App

Allows users to:

* Add notes
* Store notes in Local Storage
* Delete notes
* Automatically load saved notes after refreshing the page

### Product Listing

Displays products with:

* Category filtering
* Price sorting
* Responsive product cards

### Contact Form

Performs client-side validation by checking:

* Empty fields
* Valid email format
* Success message after submission
